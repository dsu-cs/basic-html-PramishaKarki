What is HTML?

HTML also known as Hypertext Markup Language is the standard markup language to create web pages. It consists of series of elements like headings, images, tables, links and paragraphs. HTML is used to describe what purpose our content has and how it relates to other elements on the page. The content is defined by using tags and most of them are used for layouts.

Common Tags used

  • Headings: h1, h2, h3, h4, ......
  • Paragraphs: p
  • Links: a
  • Images: img

Example of a simple HTML page structure

HTML documents are split into two main sections: the head and the body. The head contains meta information about the document, such as the title and character encoding, while the body contains the content that is displayed to users.
