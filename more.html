HTML Elements

HTML elements are the building blocks of HTML pages. They are represented by tags, which label pieces of content such as "heading", "paragraph", "table", and so on. Browsers do not display the HTML tags, but use them to render the content of the page.

Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value". For example, the href attribute in the anchor tag specifies the URL of the page the link goes to.

HTML Comments

Comments in HTML are not displayed in the browser. They are used to leave notes or explanations within the code for developers. Comments start with <!-- and end with -->.
